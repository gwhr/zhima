# 7天免登录（同设备）PRD（2026-03-31）

## 背景
- 手机号登录依赖短信验证码，频繁登录会带来短信消耗与体验成本。
- 目标是在同一设备上保持登录态，减少重复验证码请求。

## 目标
- 用户登录成功后，同一设备 7 天内免登录。
- 访问登录/注册页时，若会话仍有效，自动跳转到业务页。

## 范围（P0）
- NextAuth 会话有效期统一调整为 7 天（JWT + Session）。
- `/(auth)` 路由组增加已登录拦截：
  - 管理员跳转 `/admin`
  - 普通用户跳转 `/dashboard`

## 规则
- 仅在“同设备 + 会话未过期”情况下免登录。
- 7 天到期后，用户需重新登录（手机号登录时会再次使用短信验证码）。
- 手动退出登录后立即失效，不再免登录。

## 验收标准
- 用户登录成功后关闭浏览器再打开，7 天内访问业务页无需再次登录。
- 7 天内访问 `/login` 或 `/register`，会被自动重定向到对应业务页。
- 超过 7 天或主动退出后，访问业务页会回到登录页。
