# 登录页协议勾选 PRD（2026-03-28）
## 1. 背景
- 登录页缺少用户协议与隐私政策确认入口，存在合规与审查风险。

## 2. 目标
- 在登录环节明确展示《用户协议》《隐私政策》。
- 用户未同意前，不允许提交登录请求。

## 3. 范围（P0）
- 登录页（账号登录、手机号登录）新增协议勾选区。
- 未勾选时禁止：
  - 账号密码登录提交
  - 手机验证码发送
  - 手机验证码登录提交
- 新增可访问页面：
  - `/terms` 用户协议
  - `/privacy` 隐私政策

## 4. 验收标准
- 登录页可见协议勾选与两个协议链接。
- 未勾选时按钮禁用或提示“请先阅读并同意协议”。
- 点击协议链接可正常打开对应页面。
