## 2. 新电脑启动步骤（必须按顺序）

### 2.1 拉代码

```bash
git clone https://github.com/gwhr/zhima.git
cd zhima
git checkout main
git pull origin main
```

### 2.2 安装依赖

```bash
pnpm install
```

### 2.3 环境变量

在本地创建并填写：

1. `.env.local`（开发）
2. 如需对照可参考 `.env.example`

最少保证以下配置可用：

- `DATABASE_URL`
- `REDIS_URL`
- `NEXTAUTH_SECRET`
- `NEXTAUTH_URL=http://localhost:3000`
- `ZHIPU_API_KEY`
- `CODE_GEN_MODEL_ID`
- `THESIS_GEN_MODEL_ID`

风控新增配置（可先用默认）：

- `DEFAULT_USER_TASK_CONCURRENCY_LIMIT=1`
- `TASK_FAILURE_RETRY_LIMIT=2`
- `SINGLE_TASK_TOKEN_HARD_LIMIT=240000`

### 2.4 启动基础服务

```bash
docker compose up -d postgres redis
```

### 2.5 同步数据库（这步必须做）

```bash
pnpm db:push
```

### 2.6 启动应用与 Worker（两个终端）

```bash
# 终端1
pnpm dev

# 终端2
pnpm worker:dev
```
