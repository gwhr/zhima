# 手机号注册登录 - 功能规格

## 注册流程

1. 用户输入手机号
2. 点击获取验证码（60秒冷却）
3. 输入6位数字验证码
4. 注册成功，自动生成昵称（智码用户_XXXX）
5. 自动登录进入系统

## 登录支持

- 手机号 + 验证码登录
- 邮箱 + 密码登录
- 两种方式并存，用户自由选择

## 验证码规则

- 6位纯数字
- 5分钟有效期
- Redis 存储
- 60秒发送冷却
- 防刷限制：同一手机号每天最多10条

## API 接口

### POST /api/auth/send-code

发送验证码到指定手机号。

请求体：
```json
{ "phone": "13800138000" }
```

响应：
```json
{ "success": true, "message": "验证码已发送" }
```

### POST /api/auth/verify-code

校验验证码，完成注册或登录。

请求体：
```json
{ "phone": "13800138000", "code": "123456" }
```

响应：
```json
{ "success": true, "user": { "id": "...", "name": "智码用户_1234" } }
```

## 短信服务

- 服务商：短信宝 (smsbao.com)
- 开发阶段：使用模拟模式（验证码打印在终端）
- 生产环境：调用短信宝 HTTP API 发送真实短信

## 补充

- 注册成功后用户可选设置密码（用于后续邮箱+密码登录）
