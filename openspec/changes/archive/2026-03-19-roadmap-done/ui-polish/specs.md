# UI美化 - 功能规格

## 首页营销页

- Hero Banner：标题 + 副标题 + CTA按钮
- 功能展示：3-4个核心卖点，图标+标题+描述
- 定价表：免费版 / 专业版对比
- FAQ：常见问题折叠面板
- CTA按钮：引导注册

## 登录/注册页

- 左右分栏布局
- 左侧：品牌图 + 标语
- 右侧：登录/注册表单

## Dashboard 布局

- 顶部导航栏：Logo + 用户头像菜单
- 左侧边栏：工作空间列表 + 快捷操作（新建工作空间）
- 主内容区：工作空间卡片/列表

## 工作空间详情页

- 双栏布局
- 左侧：文件树 / 对话记录
- 右侧：代码预览 / 文件内容

## 全局样式

- 统一配色方案：主色蓝色系
- 统一字体、圆角、阴影、间距
- 过渡动画

## 响应式适配

- 移动端友好
- 侧边栏可折叠
- 表格横向滚动
