# Proposal: Free-tier Gating, Download Conversion, and Support Contact Surface

## Why

Phase-1 go-live strategy requires:

1. lowering first-use friction for new users
2. preserving clear conversion moments for paid unlock
3. exposing human support channel for high-intent users

Product copy and funnel behavior need to be consistent with "毕设辅导/助手" positioning.

## Scope

1. Keep free users able to create workspace and preview key outputs.
2. Gate full artifact download by recharge entitlement.
3. Add support contact block (title/description/QR) controllable from admin config.
4. Upgrade marketing homepage messaging to highlight assistant/co-pilot positioning and conversion value points.

## Out of scope

1. payment channel integration implementation details
2. token pricing algorithm changes (separate archive)
