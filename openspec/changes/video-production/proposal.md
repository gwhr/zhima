# video-production: 短视频自动生产

## 目标
自动生成产品推广短视频。录屏产品操作演示→AI 生成解说词→TTS 配音→FFmpeg 合成视频。

## 技术方案
- **录屏**：Puppeteer 自动操作产品页面 + 录屏
- **解说词**：AI 生成视频脚本
- **配音**：Edge TTS
- **合成**：FFmpeg 将录屏与配音合成最终视频

## 交付物
- 产品页面自动操作与录屏
- AI 解说词生成
- TTS 配音
- FFmpeg 视频合成
- 视频生成 API
- Worker 任务处理器
